# 📊 Sales Dashboard (Excel)
Excel sales dashboard  with interactive slicers and charts to analyze sales performance by region, product, and sales person,

##🔍 Overview
This project is an interactive sales dashboard created using Microsoft Excel to analyze sales performance.

## 🛠 Tools Used
- Excel
- Pivot Tables
- Charts
- Slicers

## 📈 Features
- Filter data by Region, Product, and Sales Person
- Monthly sales analysis
- Product-wise and region-wise performance
- Interactive dashboard using slicers
